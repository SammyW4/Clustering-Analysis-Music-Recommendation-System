# INST-414-Module-4-Assignment
This project uses K-Means clustering to group songs based on audio features.

The goal is to analyze different types of Drake-related songs using features like energy, danceability, and tempo.

A filtered dataset (data_sample.csv) is included due to file size limitations.

Steps included:
- Data preprocessing
- Feature scaling using StandardScaler
- Elbow method to select k
- K-Means clustering
- Cluster interpretation
